1. What is DOM?

DOM is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects. That way, programming languages can connect to the page.

An API is a set of functions and procedures that allow the creation of applications that access the features or data of an operating system, application, or other service.

3. What is Semantic HTML?

22. Speak about promises using the example of fetch API.

Promises are used to handle asynchronous operations in JavaScript. They are used to handle multiple asynchronous operations where callbacks can create callback hell leading to unmanageable code. A promise has 3 states: pending, fulfilled, rejected. A promise is created using the new Promise constructor which takes a function as its argument with two parameters resolve and reject. The resolve function is called when the promise is fulfilled and the reject function is called when the promise is rejected. The then() function is used to handle the fulfilled promise and the catch() function is used to handle the rejected promise.

26. Speak about the box model.

The box model is used to represent the HTML elements as boxes. It consists of the content, padding, border, and margin. The content is the actual content of the element. The padding is the space between the content and the border. The border is the border of the element. The margin is the space between the border and the adjacent element.
